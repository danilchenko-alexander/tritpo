# Проверка тестовых сценариев

### Представление результатов

Предоставление результатов требуется описать в следующем виде:

1. Идентификатор
2. Назначение
3. Сценарий
4. Ожидаемый результат
5. Фактический результат
6. Оценка

Тестовые сценарии:
### Прохождение теста с возможностью узнать свой результат

1. 001
2. Прохождение теста с возможностью узнать свой результат
3. Произведите следующие шаги:
	1. Запустите приложение.
	2. Нажмите кнопку Start Typing Test.
	3. Вводите слова выделенные в заполненной строке изначально светло-серым цветом  в пустую строку.
	4. После завершения печати слова нажимайте пробел, корректно введенные слова будут закрашиваться в зеленый цвет, в противном случае в красный.
4. В конце теста (по истечению 60 сек) на экране появится результат в виде: кол-во корректно введенных слов ,кол-во ошибок, процент правильно введенных.
5.После прохождения теста появилось окно с результатами тестирования.
6. 100%.
	
### Возможность просмотра таблицы результатов

1. 002
2. Просмотр результатов тестирования
3. Произведите следующие шаги:
	1. Запустите приложение.
	2. Нажмите кнопку Check Result.
4. Откроется новое окно «таблица результатов».
5.Открылось окно «Таблица результатов».
6. 100%.

### Возможность рестарта теста без занесения результатов в таблицу
	
1. 003
2. Проверка возможности рестарта теста без занесения результатов в таблицу
3. Произведите следующие шаги:

	2. Нажмите кнопку Start Typing Test.
	3. Нажмите кнопку Restart.
4. Окно теста обновится, результаты сбросятся в 0.
5. Окно теста обновилось, данные небыли записаны в таблицу.
6. 100%.
	
### Возможность очистки таблицы результатов
	
1. 004
2. Очистка таблицы результатов.
3. Произведите следующие шаги:
	1. Запустите приложение.
	2. Нажмите кнопку Check Results.
	3. В открывшемся окне нажмите кнопку Clear.
	4. В окне подтверждения нажмите кнопку Ok.
4. Таблица полностью очищена.
5. Таблица полностью очищена.
6. 100%.
	
### Возможность узнать о возникновении ошибок в ходе печати
	
1. 005
2. Уведомления об ошибках в ходе печати.
3. Произведите следующие шаги:
	1. Запустите приложение.
	2. Нажмите кнопку Start Typing Test.
	3. Введите произвольные символы в пустой строке, не совпадающие с подстрокой выделенного слова в заполненной (тестовой) строке.
4. Слово в тестовой строке будет обрамлено красной подсветкой.
5. Слово в тестовой строке приняло красную подсветку
6. 100%
	
### Возможность видеть в тестовой строке результат введенного (верно/неверно) слова

1. 006
2. Результат введенного слова.
3. Произведите следующие шаги:
	1. Запустите приложение.
	2. Нажмите кнопку Start Typing Test.
	3. Введите символы в строку.
	4. Нажмите пробел.
4. В случае корректно введенного слова его буквы закрасятся в зеленый цвет, в противном случае в красный.
5. Слова закрашиваются в соответствии с ожидаемым результатом цвета.
6. 100%.

3. Произведите следующие шаги:

### Проверка конечности количества слов в тесте

1. 008
2. Проверка конечности количества слов в тесте
3. Произведите следующие шаги:

	2. Пройдите тест несколько раз.

4. В тестовой строке всегда есть слова.
5. Слова подгружаются из файла по мере ввода последнего в тестовой строке.
6. 100%
